# Las Seis Esposas de Enrique VIII

## Contenido
- `index.html`: Página principal con la biografía de Enrique VIII y detalles de cada esposa.
- `styles.css`: Archivo de estilos para dar formato y diseño a la página.
- Imágenes y referencias históricas incluidas en el documento, incluidas por 'src'.
  
## Características
- Biografía resumida de Enrique VIII.
- Sección dedicada a cada una de sus seis esposas:
  - Catalina de Aragón  
  - Ana Bolena  
  - Juana Seymour  
  - Ana de Cléves  
  - Catalina Howard  
  - Catalina Parr  
- Imágenes ilustrativas y citas históricas.
- Referencias bibliográficas con enlaces a fuentes confiables.
